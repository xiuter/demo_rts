# 疆域铸造 H5 Demo

一款以金币资源管理为核心的手机竖屏即时策略游戏 Demo。

## 玩法

- 在己方领土相邻的六边格建造金矿、兵营和防御塔。
- 升级金矿提高收入，升级兵营训练更强单位，升级防御塔提高伤害。
- 花费金币将步兵加入训练队列，单位会自动行军、占地和战斗。
- 摧毁赤焰军团的主城即可获胜。

## 本地运行

```bash
npm install
npm run dev
```

打开 <http://localhost:4173>。

## 验证

```bash
npm test
npm run test:e2e
npm run build
```
